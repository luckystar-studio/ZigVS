This template starts in Folder Mode using build.zig.

After the template is created, ZigVS opens the generated folder and removes the temporary .zigproj file automatically.

If build.zig is not already the startup item, select build.zig in Solution Explorer, open the context menu,
and choose "Set as Startup Item."

Keep build.zig and build.zig.zon if you want to use Zig's native build workflow.
